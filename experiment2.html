Reading notes

  1. Decision rule (pre-registered): switch the default landing only if an arm beats LE on first-session activation at Bonferroni α=0.0167 at the day-12 look. A D1/install win without an activation win spawns a follow-up experiment; it does not switch the landing.
  2. Repeat-session metrics use the strict returner standard (≥30 min gap or next IST day) — naive ≥2-sessions counts are 40–50% quick re-opens and invert conclusions (Jul-11 deep-dive).
  3. Landing differs ONLY on the very first session. Return sessions boot last-watched for every arm, and the health-gated sort is identical everywhere — repeat metrics measure downstream effects of the first impression, not an ongoing treatment.
  4. Top-10 watch share is the sort's consumption fingerprint (fse_v1 returners: reorder arm 76% vs control 26%) — context, not a decision metric.
  5. Multiplicity: 3 primary comparisons at α=0.0167 (family 0.05); all repeat/commitment rows are exploratory at face-value α=0.05.
